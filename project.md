# Kraft GmbH — Проект

## О клиенте

**Компания:** Kraft GmbH  
**Сайт:** kraft-partner.com  
**Город:** Мюнхен, Германия  
**Опыт:** 27 лет (основана ~1997)  
**Слоган:** "Präzision ist unsere Stärke" (Точность — наша сила)

## Чем занимается

IT-консалтинг для финансового сектора:
- Консалтинг бизнес-процессов для банков и лизинговых компаний
- Внедрение и поддержка программного обеспечения (своего и сторонних вендоров)
- Миграция данных между системами
- Техническое и функциональное обучение персонала

**Собственный продукт:** линейка FLF (контроллинг, отчетность, планирование)

**Поддерживаемые системы:** NAVILEASE, LeasySOFT, CALMS2, OpenLease, comm.fleet, comm.lease, SAP ERP Leasing, Oracle Financials, eGECKO и другие

**Отрасли:** Лизинговые компании, банки, страховые компании

## Чего хочет клиент

1. **Больше клиентов** — основная цель
2. **Больше денег** — рост выручки
3. **Лучший сервис** — автоматизация, качество обслуживания
4. **Улучшить презентации перед банками** — выглядеть профессионально на переговорах

## Наш подход

Показываем 10+ конкретных решений, которые можно реализовать с помощью **вайб-кодинга** (AI-assisted development) быстро и недорого. Каждое решение напрямую связано с конкретным бизнес-результатом.

## Текущие проблемы сайта kraft-partner.com

- Устаревший дизайн (стиль 2012 года)
- Нет Call-to-Action кнопок
- Нет онлайн-форм (только email и телефон)
- Нет социального доказательства (отзывы, логотипы клиентов, цифры)
- Нет SEO / блога
- Слабая мобильная оптимизация
- Нет чатбота
- Нет клиентского портала
