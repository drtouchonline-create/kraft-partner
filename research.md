# Kraft GmbH — Анализ конкурентов

## Ключевой вывод

**Kraft GmbH — единственный специалист по лизинговому ПО в Германии.**  
Ни один из 10 проанализированных конкурентов не имеет глубокой экспертизы в NAVILEASE, LeasySOFT, CALMS2, OpenLease, comm.fleet, comm.lease одновременно. Это самая сильная и защищенная позиция Kraft.

**Универсальная слабость ВСЕХ конкурентов:**
- Ни у одного нет чатбота
- Ни у одного нет клиентского портала с трекингом проектов
- Ни у одного нет интерактивных инструментов лидогенерации
- Почти никто не имеет SEO-контента под узкие отраслевые запросы

### 1. BearingPoint
**Сайт:** bearingpoint.com | **Размер:** 10 000+ чел.

**Услуги:** SAP-консалтинг, Lease & Rent for S/4HANA (собственный продукт), банки, страховки, автомобили  
**Клиенты:** 5 крупнейших банков Германии, 32 из 40 компаний DAX

**Сильные стороны:**
- Единственная крупная компания с продуктом именно для лизинга (Lease & Rent for S/4HANA)
- Масштаб: 2 100+ SAP-консультантов, 78 стран
- 10+ кейс-стади на сайте
- Сайт на 20+ языках

**Слабые стороны:**
- Работают только с крупным бизнесом (DAX-компании)
- Их лизинговый продукт — только для SAP S/4HANA, не подходит клиентам на NAVILEASE, comm.lease и т.д.
- Нет чатбота, нет клиентского портала
- Недоступная ценовая модель для среднего бизнеса

**Разрыв для Kraft:** BearingPoint не обслуживает средний рынок и не знает non-SAP лизинговых систем.

## Рыночные разрывы — что Kraft может занять первым

1. **Позиция "Лизинговый IT-специалист №1 Германии"** — ни один конкурент не может претендовать на это
2. **Первый чатбот в нише** — ни у одного конкурента нет, первый получает преимущество
3. **DORA/NIS2 compliance для лизинговых компаний** — регуляторика создает срочность ("вы обязаны обновить IT")
4. **SEO под нишевые запросы** — "NAVILEASE Migration Partner", "Datenmigration Leasinggesellschaft" — конкуренции почти нет
5. **Кейс-стади с цифрами** — даже 3 анонимных кейса выводят Kraft вперед всех boutique-конкурентов
6. **Контент-маркетинг под узкие запросы** — никто не пишет статьи для лизинговых компаний
7. **Клиентский портал** — ни у кого нет, огромный WOW-фактор для существующих клиентов
8. **Средний рынок** — крупные игроки (BearingPoint, Wavestone) работают с DAX-компаниями, малый и средний лизинг не охвачен
